# pytorch_cifar
Test bed for cifar 10 and 100 datasets using PyTorch

```
pip install runstats
```

## Results
ResNet original paper: 92.63%, ResNet32, 200 epochs
SuperConvergence:
    94%, 30 epochs
    94%, 18 epochs with test time augmentations
